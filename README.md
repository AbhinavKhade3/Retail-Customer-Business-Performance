**This project focuses on analyzing retail customer behavior and overall business performance to uncover actionable insights that help improve sales, customer retention, and profitability.
The analysis transforms raw retail data into meaningful metrics and visual insights useful for business decision-making.
Objectives

*Analyze customer purchasing patterns

Evaluate business performance using key KPIs
Identify high-value customers and top-performing products
Understand revenue trends and operational efficiency
Provide data-driven recommendations for growth

*Tools & Technologies

SQL – Data extraction, cleaning, and analysis
Power BI – Interactive dashboards and visual storytelling
Excel – Data preprocessing and validation
Git & GitHub – Version control and project management

*Key KPIs Analyzed

Total Revenue
Average Order Value (AOV)
Customer Lifetime Value (CLV)
Repeat Purchase Rate
Sales by Product Category
Monthly / Quarterly Sales Trends
Customer Segmentation Performance

Certain product categories consistently outperform others

Repeat customers generate higher average order value
